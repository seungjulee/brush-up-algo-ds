Is it a BST? Coding

QUESTION DESCRIPTION

This is a very well known interview problem: Given a Binary Tree, check if it is a Binary

Search Tree (BST). A valid BST doesn't have to be complete or balanced.

Duplicate elements are not allowed in a BST.

Solution: O(N) time: http://articles.leetcode.com/2010/09/determine-if-binary-tree-is-
binary.html

(Suggested Time: 30 minutes)
